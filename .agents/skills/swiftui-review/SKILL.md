---
name: swiftui-review
description: Review SwiftUI code for best practices, maintainability, accessibility, performance, state ownership, previews, and modern API usage. Use when working on SwiftUI views, navigation, state flow, refactoring, or when the user asks for Swift best practices.
---

Review checklist:
1. Check state wrappers and ownership.
2. Check view size and composition.
3. Check accessibility and Dynamic Type.
4. Check previews.
5. Check modern SwiftUI API usage.
6. Suggest the smallest safe refactor first.
